# 🏋️‍♂️ 짐뱃 (GymBet)

> **"친구들과 헬스장 출석 및 개인 운동 목표를 걸고, 매일의 작은 목표에 베팅하는 챌린지 서비스"**

## 📌 서비스 소개
짐뱃(GymBet)은 헬스장에 등록했지만 혼자서는 꾸준히 가기 힘든 20~30대 헬스 초보자를 위한 소셜 베팅 챌린지 서비스입니다.
지인들과 그룹을 만들어 공동의 큰 목표를 설정하고, 매일 각자의 작은 운동 목표 성공 여부에 가상 포인트를 베팅합니다. '타인의 시선'과 '소소한 보상'을 통해 작심삼일을 극복하고 꾸준한 운동 루틴을 형성하도록 돕습니다.


## 🛠 기술 스택 (Tech Stack)

### Backend

- **Language & Framework**: Java21, Spring Boot
- **Database**: MySQL
- **Infrastructure**: Docker , AWS EC2, GitHub Actions CI/CD
- **Document**: Swagger
